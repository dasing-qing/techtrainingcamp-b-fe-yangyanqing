# 简易版头条搜索
### 简单实现了头条搜索的部分功能
## 功能实现
1. 功能一  
- 搜索框：支持用户手动输入词、并基于ajax实现推荐词功能。
- 结果页：通过ajax获取搜索结果数据完成列表页渲染
2. 功能二
- 实现swiper对搜索结果页tab横向切换，并根据特殊词汇（'vue', 'react', 'webpack', 'npm'，'新生训练营'）弹出
“彩蛋动画”
